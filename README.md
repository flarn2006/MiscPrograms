# Keyboard controls for `graph.c`

* Arrow keys: Pan view
* `S`: Toggle slope display
* `T`: Enter trace mode
* `Q`: Quit

## Trace mode

* Left/Right: Move cursor
* Enter: Type in X coordinate
* `T`: Exit trace mode
